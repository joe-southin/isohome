# Brief: [Project / Feature Name]

> **Purpose**: Captures *what* we're building and *why*. Target: under 300 words.
> If you're writing data models, function signatures, or tech choices, you've drifted
> into spec territory — move those to SPEC.md.
>
> **How to use**: Feed this back to Claude with:
> _"Read BRIEF.md, then help me write SPEC.md"_
